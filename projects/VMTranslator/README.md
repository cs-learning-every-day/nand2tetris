### 虚拟机

Run
有Sys.vm文件运行 

将整个目录下的.vm生成一个filename.asm ``go run ./entry/main directory filename`` 

无Sys.vm运行以下两个

``go run ./entry/main directory`` 将整个目录下的每个tmp.vm生成tmp.asm


``go run ./entry/main file`` 将file.vm生成file.asm
